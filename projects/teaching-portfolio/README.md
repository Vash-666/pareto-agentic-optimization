# Teaching Portfolio Framework

A structured system for creating and publishing video scripts, tutorials, and reference materials about multi-agent AI systems. Built on the Pareto Agentic Optimization methodology — the same quality-controlled pipeline used to build and deploy production AI workflows.

## What This Is

The Teaching Portfolio Framework provides:

- **Script templates** — STAR-format video scripts for technical content
- **Topic outlines** — Structured coverage of multi-agent concepts, quality assurance, and automation
- **Reference guides** — Detailed explanations of scoring systems and architecture patterns
- **Quality gates** — Every piece of content passes a deterministic quality check before publication

This is a tool for people who build AI systems and need to explain how they work — to colleagues, students, hiring managers, or the open-source community.

## Connection to Pareto Agentic Optimization

This portfolio is a direct product of the same system it teaches. The multi-agent architecture, quality equation, pipeline orchestration, and model-switching protocol described in these scripts are the actual mechanisms used to produce this content. Every script and guide in this directory was created, reviewed, and published through a deterministic pipeline that enforces quality thresholds at each milestone.

| Concept | Implementation |
|---|---|
| Quality gates | Every file must score ≥85/100 before publication |
| Multi-agent system | 6 specialized agents collaborate on content creation |
| Deterministic scoring | Quality = (Prompt × 0.65) + (Memory × 0.20) + (Model × 0.10) + (Tools × 0.05) |
| Pipeline orchestration | 1 command runs 5 phases across 7 tools |
| Context preservation | 100% context maintained across model transitions |

## Quality Thresholds

Content at each milestone must pass these minimum scores:

| Milestone | Minimum Score | What It Means |
|---|---|---|
| Start | 60/100 | Structure exists, all files present |
| Review | 75/100 | Content is accurate, readable, recruiter-ready |
| Complete | 85/100 | Publication quality, fully verified |
| Production | 90/100 | Enterprise-ready, independently audited |

Scoring is deterministic — not subjective. The quality equation weights prompt quality, memory usage, model selection, and tool integration.

- **AI engineers** — Learning multi-agent architecture patterns and quality control

## How to Use This Framework

1. **Pick a topic** from the topic outlines or scripts
2. **Create content** using the concept or demo templates
3. **Run the quality gate** to verify the content meets threshold
4. **Publish** through the pipeline to GitHub

All content created through this framework inherits the same quality guarantees as the system that builds it.
